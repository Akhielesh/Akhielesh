## 2024-03-11 - Server Component Layout Extraction
**Learning:** Next.js allows importing Client Components (e.g., `ShineCard`, `Reveal`) directly into Server Components. Marking large layout sections (like `HeroSection`, `FeaturedAiProducts`, and `ContactSection`) with `"use client"` just because they contain a few interactive Client Components forces Next.js to send all the static layout markup and deeply nested data (e.g., case studies mapped into text) to the browser.
**Action:** When a layout component mostly maps static content and only has isolated interactive parts, ensure it does not use `"use client"`. Let the nested components (like `Reveal` or `ShineCard`) be the Client Components. This drastically reduces JavaScript bundle size and hydration overhead without losing interactivity.
